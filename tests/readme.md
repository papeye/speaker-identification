> [!NOTE]  
> We are using [pytest](https://docs.pytest.org/en/stable/) for testing


# Running tests

0. Install pytest
```
pip install pytest
```

1. Run tests
```
pytest tests
```
> [!IMPORTANT]  
> pytest automatically finds files named test_*.py or *_test.py (we are taking the former as a convention)
> (not that * cannot contain underscore "_")
